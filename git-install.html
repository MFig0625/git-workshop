Installing git

Command Line Cheat Sheet

cd /path/to/file
change directory
cd ~ or cd
return to home directory
mkdir
make directory

Mac

Type the following and you should see the following:

$ which git
/usr/bin/git
$ git --version
git version 1.9.3 (Apple Git-50)

Whoa? What's this dialogue box?

It's okay. click install. :)

Windows

Visit git-scm.com.

Download and install the software as you normally would.

If you are prompted to install git-bash, yes, install it.

Configure git

$ git config --global user.name "your name"
$ git config --global user.email "your email"

You can check your settings by typing:

$ git config --list

Configuring your keychain

Mac OS X

$ git credential-osxkeychain

Windows

$ git config --global credential.helper wincred

git

What is git?

  • version control software
  • keeps track of 'snapshots' -- what has changed?
  • stores everything as a repository
    • files and metadata about the files
  • allows everyone to access all files

What is GitHub?

  • github.com
  • a service for hosting git repositories
  • free for open-source projects
  • small fee for private repo hosting

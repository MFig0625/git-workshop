Git Cheat Sheet

git --help
Help!
git status
Your best friend: It tells you everything
git add . or git add filename
Add file(s) to your repo
git commit -m "your commit message"
Commit the files
git push -u origin master
First time you push to GitHub
git push
Every push after
git log
See a repo's commit history
git diff filename
See what changes you've made to an uncommitted file
git checkout filename
Revert to the last (unchanged) commit.
git reset --hard [commit number]
Changes back to the specified commit
